# punk_os

PunkOS is an application written in Flutter, running on Android.

PunkOS is not a normal application, it much likes a Operate System. Because I plan to develop many applications inside PunkOS.

In my plan: PunkOS is a open platform, by using low-code tech, every one can develop her/his app themselves.
